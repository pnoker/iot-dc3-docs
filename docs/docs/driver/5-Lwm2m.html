Lwm2m & Coap 驱动 #

功能列表 #

  • 读取资源
  • 写入资源
  • 订阅资源
  • 取消订阅
  • 执行程序
  • 注册监听
  • 注册认证
  • 标准资源

基础说明 #

位置：iot-dc3\dc3-driver\dc3-driver-lwm2m

Lwm2m：Lwm2m是一种基于CoAP协议的物联网协议，用于连接设备和云端，实现设备管理、设备数据采集、设备控制等功能。

相关配置项: 详情

快速开始 #

确保已经配置好了基础开发服务

  1. 启动驱动

  1. 配置模板
模板的意思是该设备有哪些属性可以采集,比如温湿度传感器可以采集温度,湿度

  1. 新增设备
  2. 配置设备位号属性
位号属性的意思是该设备的该属性通过哪个值去采集,例如在demo中,温度属性的采集路径为/3303/0/5700

  1. 启动设备

找到/resource/lib/leshan-client-demo.jar
java -jar leshan-client-demo.jar -n 平台设备id -u 服务器地址(不指定则默认localhost:5683)

  1. 查看数据
