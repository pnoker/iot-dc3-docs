快速开始 #

确保已经配置好了基础开发服务

1.启动驱动

2.添加数据模板

驱动自带的业务代码中有海拔[4]+速度[8]+液位[8]+方向[4]+锁定[1]+经纬[21] 6个业务属性,此处只配置了一个

3.添加设备

此处设备名称将在报文中出现,请尽量使用纯英文或数字,或自行替换代码中的部分逻辑

4.配置位号属性

起始字节: 读取的起始字节位置

结束字节: 读取的结束字节位置,非可变长属性可忽略配置

类型: 数据真实数据类型

关键字: 业务逻辑,见代码

本示范中温度的起始字节位置为23,标准4字节,所以结束字节可忽略配置

5.发送数据

使用网络调试助手或各类TCP测试工具发送报文

6.查看数据
