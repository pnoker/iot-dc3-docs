边缘网关 驱动 #

TIP

该驱动本质上是一个MQTT驱动,参考MQTT 驱动,后续会完善部分业务逻辑达到开箱即用

介绍 #

边缘网关是一种特殊的设备，它直接通过标准MQTT协议接入平台（Iot-dc3)。边缘网关下面可以接入子设备，子设备的认证信息由平台生成并且下发给网关，子设备的所有登陆登出、数据上报和接受指令下发，都是通过边缘网关发送给平台。子设备和网关之间的通信协议或方式可以有多种，例如串口、TCP、UDP、蓝牙等。
