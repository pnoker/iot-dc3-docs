构建 IoT DC3 Web UI #

git clone https://github.com/pnoker/iot-dc3-web.git
cd iot-dc3-web

#这步至关重要, 推荐使用 yarn
npm install -g yarn --registry=https://registry.npm.taobao.org
yarn

启动 IoT DC3 Web UI #

yarn serve

访问 IoT DC3 Web UI #

  • 访问 http://localhost:8080 (opens new window) 即可进入登陆页面！
