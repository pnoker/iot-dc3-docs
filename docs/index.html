DC3是一个基于Spring Cloud的开源的、分布式的物联网(IOT)平台,用于快速开发物联网项目和管理物联设备,是一整套物联系统解决方案。

开放无界

采用 Apache 2.0 许可，代码完全开源，持续迭代和改进，始终反映最新的技术和用户需求。

万物互联

广泛的物联网协议：OPC、PLC、Modbus、MQTT、TCP、UDP、CoAP、LoRa 等，支持快速自定义和拓展协议。

智慧前瞻

基于强大的 Spring Cloud 技术栈，轻松构建高可用、可扩展、分布式、云原生的应用程序。
