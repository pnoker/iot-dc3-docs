常见问题 #

1.IoT DC3 平台目前支持哪些驱动协议？ #

广泛的物联网协议：OPC、PLC、Modbus、MQTT、TCP、UDP、CoAP、LoRa 等，支持快速自定义和拓展协议。

2.IoT DC3 平台目前有 Demo 演示地址吗？ #

计划 5 月底上线 Demo，敬请关注。

你也可以通过以下方式快速启动，详情请阅读 🚀 快速体验

# 下载iot-dc3源码
git clone https://gitee.com/pnoker/iot-dc3.git
cd iot-dc3/dc3/demo
# 启动容器
docker-compose up -d

3.IoT DC3 平台有商业版本吗？ #

我们不提供专门的商业版本，但我们可以接受有偿的定制开发，以满足客户的特定需求。

我们的代码采用 Apache 2.0 许可协议，完全开源，以便于社区的共同参与和持续改进。我们一直保持敏锐的技术洞察力和对用户需求的高度关注，不断迭代和更新我们的产品。

4.怎么获取 MySQL 和 MongoDB 数据库初始化脚本？ #

您好，MySQL 和 MongoDB 数据库的初始化脚本在 dc3/sql 目录下。

  • sql
    • mysql
      • iot-dc3.sql ：IoTaDC3 平台 MySQL 主表
      • iot-dc3-nacos.sql ：Nacos 服务注册中心主表
    • mongo
      • iot-dc3.js ：IoTaDC3 平台 MongoDB 主表
